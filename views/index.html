Endpoints

GET user's exercise log:

GET /api/users/:_id/logs?[from][&to][&limit]

[ ] = optional; from-to = dates (yyyy-mm-dd); limit = number;

POST new user:

POST /api/users

POST new exercise:

POST /api/users/:_id/exercises
